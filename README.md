## Dataset

### MCMD-DroneVehicle

We construct **MCMD-DroneVehicle** based on the registered infrared-visible image pairs and detection annotations of the DroneVehicle dataset. The benchmark introduces a **Modality-Coupled Multi-Degradation** protocol for drone infrared-visible image fusion.

Different from conventional degraded fusion datasets that mainly degrade a single modality or treat two modalities independently, MCMD-DroneVehicle jointly considers visible-specific degradations, infrared-specific degradations, and physically plausible cross-modal degradation coupling.
